A little about it

Web design refers to the design of websites that are displayed on the internet. It usually refers to the user experience aspects of website development rather than software development. It encompasses many different skills and disciplines in the production and maintenance of websites. The different areas of web design include web graphic design; user interface design (UI design); authoring, including standardised code and proprietary software; user experience design (UX design); and search engine optimization

These are somethings that I Learnt



Flexbox

aims at providing a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown and/or dynamic I used Flexbox to arrange the flow of several layouts



Grid Layout

Its a two-dimensional grid-based layout system that aims to do nothing less than completely change the way we design grid-based user interfaces This site has several uses of grid layout like for example the Photo Gallery the content sections of the individual games pages are all designed using the CSS Grid Layout



Animations

Animations are an efficient way to catch eye attention and interact with the user actions. Through out the entire web page I have used several such animations like fade in and fade out or roll in and roll out or an entire carasoul using only css animaion. Its not just CSS that is used to create these great animations but also an extensive part of these animations has been created using vanilla JavaScript. Like for example the fading in of the content sections in the individual games pages when the pages first loads or the loader itself are all created using JavaScript



Adobe Photoshop

Adobe Photoshop is a raster graphics editor developed and published by Adobe Inc. for Windows and macOS. Photoshop can be used to edit and compose raster images in multiple layers and supports masks, alpha compositing and several color models including RGB, CMYK, CIELAB, spot color, and duotone. I learnt to use Photoshop and edit pictures that I took from over the internet merging them, cropping them etc in order to use them in the website Like for example the poster with image map was created by merging photos together in photoshop. While the gallery photos were edited into uniform sizes for the purpose of using it in grid
